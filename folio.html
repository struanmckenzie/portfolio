University Assignments

These programs are ones I made for assignments in my computer science degree that I deem worthy of showing off what I can do. Each project has some form of detailed documentation you can view via a links, this page only has brief descriptions.

Project Nova

This was my final project for the Introduction to Software Development module. It's built in Java for the command line, I made it look as pretty as I could without having to make a GUI which I didn't know how to use. The game itself is effectively battle-ships but you have power ups and booby-traps to deal with and has a surprisingly long play time. The creatures and traps are randomly placed, the game state can be saved and loaded from text files. Grade received: A2 (104/110)
Project Nova - GitHub

Memory Game

I had so much fun developing this one as part of a team!! We had a very open brief:

"...create an embedded system that demonstrates what you’ve learned in the module so far... ...the embedded system you create can take any form you like."

We decided to make a memory game for it using a few Raspberry Pi Picos and MicroPython. It uses physical buttons, lights, and buzzers and all links together using an MQTT server. We also thought it would benefit from a website that updates with players' progress. Grade received: A1

Personal Projects

These are some of the projects I've worked on by myself (please take these with a grain of salt, most of them were made when I was still at high school).

Hours Calculator

This was a calculator I developed so I could calculate how much I should have been paid for my work at Dawyck Cafe. I never ended up finishing it because I didn't have faith at the time that I could make it to a standard where I trusted it would work.
Maybe in the future I'll come back to it or remake it in Java as a graphical desktop or mobile app. It was also made before I learned how Git and GitHub worked properly so all had different files to show development.
Hours-Calculator - GitHub

Inventory program

This was a program I made when I had obtained a lot of computer parts at one point and I wanted a way to document all of it. The best solution was to make a program for this in the only language I knew back then, Python, so I could organise my stuff and refine my coding skills. This program is the result of forgetting that spreadsheets exist. I uploaded the project to GitHub recently because back then I didn't see the point of using it.
inventory-program - GitHub

Band website

When I was in the band I thought since we were doing gigs and things that it would be a good idea for us to have a website. Since I was studying computing science at the time I decided to make one myself to improve my web dev skills. It would have information about us for anyone to see and would have updates on gig dates and that sort of thing, but I stopped updating it once I left the band.
Even though it looks objectively awful compared to this website I'm particularly pleased with this project because a local newspaper actually used it to write a small article about us!
